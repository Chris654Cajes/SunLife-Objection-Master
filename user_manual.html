🚀 Getting Started

Installation

  1. Download the SunLife Objection Master APK file
  2. Enable "Install from unknown sources" in your device settings
  3. Tap on the APK file to begin installation
  4. Follow the on-screen prompts to complete installation
  5. Launch the app from your app drawer

First Launch

When you first launch the app, you'll see the main screen with:

  • The app title at the top
  • A search bar for finding specific objections
  • A scrollable list of available objections
  • Each objection card shows the title and preview text

📖 How to Use

Searching for Objections

  1. Tap on the search bar at the top of the main screen
  2. Type keywords related to the objection you're facing
  3. The list will automatically filter as you type
  4. Tap on any objection to view detailed information

Viewing Objection Details

Each objection detail screen includes four key sections:

  • Response Script: The exact words you can use to respond
  • Follow-up Questions: Questions to keep the conversation moving
  • Psychology: Understanding the customer's mindset
  • Confidence Tip: How to deliver the response effectively

Using Audio Features

  1. On the main screen, tap the play button on any objection card
  2. In the detail screen, tap the play button next to the response script
  3. The app will read the response aloud using text-to-speech
  4. Use this to practice delivery or for hands-free learning

💡 Tips and Best Practices

🎯 Practice Regularly

Review objections daily to internalize responses and build muscle memory.

👂 Listen First

Always listen carefully to the complete objection before responding.

🤝 Build Rapport

Use the psychology insights to understand and connect with customers.

📚 Customize Responses

Adapt the scripts to your personal style and specific situations.

Remember: The best objection handling feels natural and authentic, not rehearsed!

⚙️ Technical Information

System Requirements

  • Android 5.0 (Lollipop) or higher
  • Minimum 2GB RAM recommended
  • 50MB available storage space
  • Internet connection for initial content download

Technology Stack

Android Jetpack Kotlin Material Design MVVM Architecture Text-to-Speech RecyclerView

Architecture

The app follows modern Android development practices with:

  • MVVM Pattern: Clean separation of concerns
  • ViewModel: Manages UI-related data and survives configuration changes
  • LiveData: Reactive data handling
  • Material Components: Consistent, modern UI elements

🔧 Troubleshooting

Common Issues

Audio Not Playing

Check your device's text-to-speech settings and ensure volume is turned up.

Search Not Working

Try restarting the app and ensure you have a stable internet connection.

App Crashes

Clear the app cache and restart your device if issues persist.

Slow Performance

Close other apps and ensure you have sufficient available storage.

Contact Support
